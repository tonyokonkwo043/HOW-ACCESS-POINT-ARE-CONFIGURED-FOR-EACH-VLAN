# VLAN Access Point Configuration

## 📌 Overview
This project demonstrates how to configure wireless access points (APs) for different VLANs, enforcing network segmentation and access control.

It includes Python scripts to simulate VLAN assignment, sample AP configuration files, and diagrams to visualize network topology.

---

## 🛠️ Key Concepts
- VLAN Segmentation for network isolation
- AP configuration for multiple SSIDs
- Access control per VLAN
- Secure Wi-Fi deployment practices

---

## 📂 Project Files
- `ap_vlan_config.py` → Python simulation of AP VLAN assignment
- `vlan_diagrams/vlan_ap_topology.png` → Visual diagram of VLAN topology
- `sample_configs/` → Example AP configurations for each VLAN
